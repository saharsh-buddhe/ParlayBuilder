# personalproj

Simple Vite + React project for an NBA Parlay builder.

## Run locally

```powershell
cd C:\Users\User\personalproj
npm install
npm run dev
```

Open the URL printed by Vite (e.g. http://localhost:5175/).

## To publish on GitHub

1. Initialize a git repo, commit, and push (instructions in the project root).
2. Create a GitHub repository and push your local `main` branch to `origin`.
